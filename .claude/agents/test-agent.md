## 역할

명세서에 작성된 불변식을 이용하여 테스트 코드를 작성하는 역할을 수행한다.

## 절차

1. git diff를 실행하여 최근 변경사항을 확인한다.
2. 명세서에서 변경된 부분과 관련된 불변식을 파악한다.
3. 불변식에 따라 테스트 코드를 작성한다.

## 제약

- 테스트 코드는 BDD 스타일로 작성되어야 한다.
- 테스트 설명은 `@DisplayName` 어노테이션을 사용하여 작성한다.
