# 유비쿼터스 언어

## 공통

> 주제: LXP
>

| 단어 | 의미 |
| --- | --- |
| LX (학습 경험) | 학습자가 단방향으로 지식을 받기만 하는 것이 아닌, active한 활동으로 양방향 소통을 하는 과정에서 능동적으로 생각하고 학습하는 과정에서 경험하는 것들이 학습 경험이다. |
| LX 유도 방법 | 강좌마다 미션을 두고, 회원의 미션 답안이 다른 회원들과 공유되도록 하여 미션 풀이와 소통 과정에서 능동적으로 학습하고 성장하는 경험을 느끼도록 유도한다. |

## 공통 목표

- 무조건 운영진의 정책대로 따라야함
- 회원 가입은 언제든지 가능
- 비회원에게는 콘텐츠 전부 비공개
- 구독권이 있어야 강좌와 강의, 미션 콘텐츠 확인이 가능함
- 회원 가입 시 1개월의 구독권을 주어 무료 체험 기간을 부여함
- 강좌 공개는 어드민 승인제
- 강좌 후기는 해당 강좌의 문제 1개 이상의 미션에 답안을 제출한 구독자만 작성 가능
- (Front) 홈페이지 첫 화면에서 미션을 우선적으로 노출한다.

## 회원 유형 구조

- 강사
- 회원
    - 구독자
    - 비구독자

## 회원

> 본 서비스를 사용하는 사용자와 사용자의 정보를 관리하는 도메인
>

| 단어 | 의미 |
| --- | --- |
| 회원 | 서비스 이용을 위해 등록한 사용자 (구독자 + 비구독자) |
| 구독자 | 서비스를 이용하기 위해 가입 + 유효한 구독권을 소유한 자 |
| 비구독자 | 서비스를 이용하기 위해 가입 + 유효하지 않은 구독권을 소유한 자 |
| 비회원 | 가입하지 않은 자 |
| 회원 정보 | 등록한 사용자의 닉네임, 패스워드 등을 포함한 개인 정보 |
| 회원 정보 수정 | 개인 정보를 수정하는 행위 |
| 강좌 | 학습 컨텐츠 |
| 미션 | 학습 컨텐츠 |
| 신고 | 회원이 다른 회원의 불온전한 행위에 대해 사이트 운영자에게 알리는 행위 |

| 단어 | 의미 |

| 신고 | 회원이 강사의 불온전한 행위에 대해 사이트 운영자에게 알리는 행위 |
| 강좌 | 자신이 창작한 컨텐츠 |
| 미션 | 자신이 창작한 컨텐츠 |
| 정지 | 강사의 모든 활동을 금지함 |

## 강좌

> 구독자가 볼 수 있는 학습 컨텐츠 (강의/미션) 들을 묶어놓은 단위 도메인
>

| 단어 | 의미 |
| --- | --- |
| 강좌 | 여러 강의를 포함하는 묶음 단위 |
| 강의 | 강좌에 속하는 개별 학습 콘텐츠 |
| 생성 | 강좌를 생성함 |
| 수정 | 강좌를 수정함 |
| 공개 | 강좌를 회원들이 볼 수 있게 함 |
| 비공개 | 강좌를 회원들이 볼 수 없게 함 |
| 삭제 | 소프트 딜리트 |
| 회원 | 구독권이 유효한 사용자 |
| 강사 | 강의가 속한 강좌를 개설한, 강사 권한을 가진 사용자 |
| 신고 | 회원이 불온전한 강좌에 대해 사이트 운영자에게 알리는 행위 |

>

| 강의 | 강좌에 속하는 개별 학습 콘텐츠 |

## 미션

> LXP핵심
>

| 단어 | 의미 |
| --- | --- |
| 미션 | 강좌에 포함된 퀴즈형 콘텐츠로 정답도 포함되어 있다 |
| 미션 생성 | 강좌에 미션을 추가함 |
| 미션 수정 | 강좌에 포함된 미션이 수정됨 |
| 미션 삭제 | 강좌에 포함된 미션이 삭제됨 |
| 미션 정답 | 미션의 내용에 대응되는 답안 |
| 구독자 | 미션을 풀 수 있는, 유효한 구독권을 가진 회원 |
| 강사 | 미션이 속한 강좌를 개설한, 강사 권한을 가진 사용자 |
| 공개 | 해당 미션이 회원에게 보이게 설정함 |
| 비공개 | 해당 미션이 회원에게 보이지 않게 설정함 |
| 신고 | 회원이 불온전한 미션에 대해 사이트 운영자에게 알리는 행위 |

## 구독권

| --- | --- |

| 단어 | 의미 |

| 미션 픽 추가 | 회원이 특정 미션을 자신의 관심 대상으로 시스템에 등록하는 행위 |
